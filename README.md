### Compile and Run
Before compiling the program, in **Project Folder**:
```console
mkdir build
```
```console
cd build
```

Then in `build/`:
```console
cmake ..
```

To compile the program, in `build/`:
```console
make
```

To run the program, in `build/`:
```console
./exec
```
